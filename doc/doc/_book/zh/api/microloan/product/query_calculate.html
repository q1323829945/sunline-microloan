查询当前产品还款计划

请求

Path /microloan/ConsumerLoan/{productId}/{amount}/{term}/calculate
Method GET
Description 查询当前产品还款计划

路径参数

参数 类型 M/O 说明
productId string M 产品编号
amount string M 金额
term string M 期限

参数 类型 说明
data object

∟schedule array 还款计划
∟∟period integer 期数
∟∟repaymentDate string 还款日期
∟∟installment decimal 分期付款
∟∟principal decimal 本金
∟∟interest decimal 利息
∟∟fee decimal 费用
